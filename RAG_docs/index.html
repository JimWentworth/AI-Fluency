What this is (and isn’t)
A viewer for the authoritative design docs. It’s not a CMS; it won’t auto-sync unless regenerated. Treat it as a “shared brain” snapshot you can hand to collaborators.
About
  • No server required: open index.html in a browser (or host on GitHub Pages).
  • Accessible: skip-link, semantic landmarks, keyboard-friendly search.
  • UIUC-aware styling: Illini colors used lightly; readability first.
