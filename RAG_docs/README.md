# RAG Docs HTML Viewer

This folder is a static HTML interface generated from the AI Fluency Certificate RAG corpus.

## How to use
- Open `index.html` in a browser (Chrome/Firefox/Safari).
- Or host this folder on GitHub Pages (recommended) for easy sharing.

## What’s included
- `index.html` — landing page + navigation
- `docs/` — one HTML page per RAG doc
- `assets/manifest.json` — doc metadata used to build nav
- `assets/search.json` — prebuilt search index (titles + content)
- `assets/site.css`, `assets/site.js` — styling and logic
