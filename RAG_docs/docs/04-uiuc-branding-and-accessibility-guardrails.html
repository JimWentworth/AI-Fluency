University of Illinois Branding & Accessibility Guardrails

Institutional design constraints (for certificate materials)

Canonical UIUC brand guidance (official)

  • Brand Guidelines (home): https://brand.illinois.edu/
  • Visual Identity — Color: https://brand.illinois.edu/visual-identity/color
  • Visual Identity — Typography: https://brand.illinois.edu/visual-identity/typography
  • Visual Identity — Logo: https://brand.illinois.edu/visual-identity/logo
  • Resources / Downloads: https://brand.illinois.edu/resources/downloads
  • UIUC Brand CDN (colors reference): https://cdn.brand.illinois.edu/colors.html

How we apply UIUC branding in this certificate (project norms)

  • Illinois branding should be recognizable but not overpower instructional clarity.
  • Use UIUC primary colors purposefully and accessibly; avoid overusing secondary palettes.
  • Prefer consistent typography and spacing rules across pages to reduce cognitive load.
  • Clearly mark what is official certificate content vs. external vendor examples.

Accessibility guardrails (non-negotiable)

  • Ensure adequate color contrast for text and key UI elements.
  • Provide alt text for images and meaningful labels for buttons/interactive elements.
  • Avoid “color-only” meaning (use icons/labels in addition to color).
  • Keep layouts responsive for mobile and varied screen sizes.

AI-generated visuals and institutional representation

  • AI-generated images/text used in official materials must be reviewed for:
    • brand consistency
    • accessibility
    • clarity and appropriateness for the audience
  • Avoid synthetic imagery that could imply official endorsements or misrepresent the university.

Notes for Canvas + DesignPLUS

Canvas rendering and DesignPLUS components may constrain typography and layout. The implementation guide (06_...) defines project defaults that align with UIUC branding where feasible.
