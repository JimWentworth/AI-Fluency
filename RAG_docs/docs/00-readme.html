AI Fluency Certificate — RAG Pack (Institutional Design Corpus)

Date: 2025-12-26
Owner: Jim Wentworth (CITL, University of Illinois Urbana-Champaign)

What this is

This folder contains the canonical “design authority” documents for building and maintaining the Generative AI Fluency Certificate. It is intended to be uploaded into a ChatGPT Project (or similar RAG workspace) to:

  • keep the certificate’s conceptual framework stable,
  • make design decisions legible to collaborators,
  • preserve rationale and provenance over time,
  • support AI-assisted drafting without losing intent,
  • guide Canvas + DesignPLUS implementation choices.

How to use (recommended workflow)

  1. Start with the Canonical Overview (01_...). Treat it as the source of truth for the learning arc and module boundaries.
  2. When making changes, record why in Design Rationale & Decisions (03_...). Keep it short, decision-focused.
  3. For any new page/activity, follow the Canvas + DesignPLUS Implementation Guide (06_...) and validate against the Example page (07_...).
  4. Use Tool Documentation References (05_...) whenever you need to verify platform capabilities.
  5. Use Branding & Accessibility Guardrails (04_...) when styling pages, selecting colors, and creating visuals.

Design norm (internal)

We document “behind-the-scenes” work. Drafts are allowed; rationale must remain visible.
If AI assists in drafting, we preserve: intent, constraints, tradeoffs, and what the human decided.
