Content & Prompt Templates

Reusable patterns for building certificate content

Canvas content page scaffold (recommended)

  • H2: Page Title
  • Intro paragraph (1 short paragraph): what learners will gain
  • H3: Why This Matters (2–3 bullets)
  • H3: Key Concepts (2–4 short explanations)
  • H3: Activity or Knowledge Check (short, actionable)
  • H3: Optional: Discipline-specific variants (faculty/staff/student)
  • Embed section (if needed): responsive iframe wrapper + instructions

Prompt template: “Design-first page drafting”

Use this when drafting a Canvas page while preserving human authority.

Prompt: You are helping draft a Canvas content page for the Generative AI Fluency Certificate. Follow this structure exactly: H2 title; intro paragraph; H3 Why This Matters (bullets); H3 Key Concepts; H3 Activity/Knowledge Check. Constraints:

  • Keep it tool-agnostic unless the page explicitly teaches a tool.
  • Use UIUC-friendly tone: clear, direct, non-hype.
  • Include at least one “boundary” sentence describing what the concept is NOT. Topic: {paste topic} Audience: {faculty/staff/students/varied} Length: ~350–600 words

Prompt template: “Document a design decision” (internal)

Prompt: Summarize the following design discussion into a decision log entry. Output:

  • Decision (1 sentence)
  • Rationale (2–4 bullets)
  • Tradeoffs (1–3 bullets)
  • Follow-ups (bullets, if any) Text to summarize: {paste thread or notes}
