Hands-On Exploration: Prompt Patterns & Persistent Settings

Activity 1: Build a Prompt Pattern

Choose a recurring task in your work (e.g., drafting assignment prompts, summarizing articles, designing quiz questions). Adapt the template below to create a reusable pattern.

You are an assistant helping me with: [type of work]. Audience: [who is this for]. Context: [course, level, discipline]. Task: [what you want done]. Constraints: - Use this tone: [tone] - Use this structure: [headings, bullets, etc.] - Follow these criteria: [rubric elements, policies, or guidelines] Before you begin, ask me 2–3 clarifying questions to avoid guessing.

Try it:

  1. Paste your adapted pattern into ChatGPT.
  2. Run it with two different content examples.
  3. Note where the structure holds and where it drifts.

Activity 2: Test Persistent Personalization

Using custom instructions or memory (if enabled), define how you would like ChatGPT to support your teaching or research work.

In future conversations, prioritize helping me with: [teaching tasks, research tasks, communication tasks]. When I ask for help: - Default to this tone: [e.g., concise, exploratory, student-friendly]. - Ask clarifying questions if my request is vague. - Flag any potential academic integrity or privacy concerns you notice.

After saving your settings, run a fresh conversation and see how the model behaves differently. Identify one benefit and one potential risk or limitation.
