Factual vs. Creative Use

Understanding when to use LLMs for facts versus creativity

Large Language Models excel at both factual tasks and creative work, but they function differently in each domain.

Side-by-Side Comparison

Factual Use Cases

Goal: Retrieve or apply existing knowledge accurately

  • Answering knowledge questions
  • Code generation
  • Data extraction
  • Mathematical calculations
  • Language translation
  • Summarization

Key: There's a "right answer"

Creative Use Cases

Goal: Generate novel, original content

  • Story writing
  • Poetry composition
  • Brainstorming ideas
  • Marketing copy
  • Character dialogue
  • Creative problem-solving

Key: Multiple valid outputs

The Hallucination Problem

Critical Limitation

What it is: LLMs can confidently generate false information that sounds plausible.

Why it happens: LLMs predict likely text patterns, not truth.

Hallucination Example

Prompt: "What did Marie Curie discover in 1923?"

LLM might say: "In 1923, Marie Curie discovered polonium-218..."

Problem: Completely fabricated!
